# S4: 投资组合月报

> **类型**：金融
> **难度**：★★★
> **建议时长**：90分钟
> **适合人群**：有资管、FOF、财富管理背景

> **📌 教学材料声明**：本任务卡使用模拟数据，所有公司名称、财务数据、市场信息均为教学虚构，不构成任何投资建议。

---

## 5分钟速用框

| 项目 | 建议 |
|------|------|
| **适用场景** | 投资组合月报、客户简报、绩效归因初稿 |
| **推荐 Context** | `data/S4-material-A.md` 组合数据 + `data/S4-material-B.md` 市场环境摘要 |
| **Harness 起点** | 只基于模拟数据；不给投资建议；不预测具体价格走势；区分本月收益率和本年收益率 |
| **最小成果** | 月度表现摘要 + 资产配置表 + 归因分析 + 风险提示 + 下月关注点 |
| **汇报重点** | 展示 Agent 是否混淆收益口径，是否只讲收益不讲风险 |


---

## 背景设定

你是某家办的资产管理部投资经理。你管理一个**多资产组合**，包含股票、债券和另类投资。每月你需要向家办首席投资官（CIO）提交一份组合月报。

本月（2024年10月）市场波动较大，你需要快速整理组合表现，并给出调整建议。


---

## 任务目标

基于提供的模拟组合数据，生成一份**投资组合月报**，包含：

1. **组合概览**（总体表现、基准对比）
2. **分项表现**（各类资产的表现对比）
3. **归因分析**（收益/亏损的主要来源）
4. **风险指标**（波动率、最大回撤等）
5. **下月展望与调整建议**


---

## 输出要求

### 格式要求

1. **组合概览**（表格：目标比例/实际比例/本月收益/本年收益）
2. **分项表现分析**（每类资产的表现 vs 基准，bullet points）
3. **归因分析**（收益/亏损的主要来源，2-3条）
4. **风险指标摘要**（表格：波动率/最大回撤/夏普比率/Beta）
5. **下月展望**（2-3条，基于市场环境）
6. **调整建议**（是否需要调仓？调哪类资产？）

### 约束要求

- 只基于提供的模拟数据
- 所有数据需标注来源（如"根据组合月报数据"）
- 不给投资建议（不输出买入/卖出/持有/目标价等建议）
- 不要预测具体的价格走势
- 调整建议需有明确的数据支撑
- 完成后请列出2个分析中的不确定性

## 建议的 Prompt 框架

```
【Context】
我们是某家办资产管理部，管理一个1.2亿元的稳健增长型组合。
已上传组合持仓数据、表现数据、风险指标和市场环境摘要。

【Objective】
基于提供的模拟数据，生成一份投资组合月报，供CIO审阅。

【Style】
专业、数据驱动，每个观点有数据支撑。

【Tone】
客观、冷静，不夸大收益，不回避亏损。

【Audience】

【Response】
1. 组合概览（表格）
2. 分项表现分析（bullet points）
3. 归因分析（2-3条）
4. 风险指标摘要（表格）
5. 下月展望（2-3条）
6. 调整建议（是否需要调仓？）

【Constraint】
- 只基于提供的模拟数据
- 数据标注来源
- 不预测具体价格走势
- 建议需有数据支撑
- 完成后列出2个不确定性
```

## 评价要点

| 维度 | 重点 |
|------|------|
| **Prompt设计** | 是否要求了归因分析和调整建议？ |
| **Context管理** | 是否综合使用了组合数据和市场环境？ |
| **Harness约束** | 是否有"不预测价格"的约束？建议是否有数据支撑？ |
| **输出质量** | 月报是否专业？CIO能否在5分钟内抓住重点？ |
| **迭代过程** | 是否发现了数据中的异常？如何处理？ |

## 提示与技巧

### 可能的"翻车"点

1. **Agent 可能混淆收益率**：把"本月"和"本年"混在一起
2. **Agent 可能给出具体预测**：如"预计茅台下月上涨5%"
3. **Agent 可能忽略风险指标**：只谈收益不谈风险

4. **如果 Prompt 只写"分析组合表现"而不指定格式**：Agent 可能输出大段叙述而非表格，CIO 无法5分钟抓住重点
5. **如果只给组合数据不给市场环境**：Agent 无法解释"为什么本月亏损"，归因分析只能写"国内股票跌了"
### 如何提升质量

- 明确要求"先讲风险，再讲收益"
- 要求"调整建议必须引用具体数据"
- 增加校验："检查本月收益率和本年收益率是否有矛盾"

### 可扩展的方向

- 增加"情景分析"（如果某只股票下跌20%，对组合的影响）
- 增加"与同类组合对比"（相对排名）
- 增加"交易成本估算"（调仓建议的成本）

## 汇报提示

汇报时建议结构：
1. **场景与目标**（30秒）：选了S4投资组合月报，目标是向CIO汇报本月组合表现
2. **Prompt 展示**（1分钟）：重点说明如何用 Response 指定表格+要点格式，如何用 Constraint 禁止价格预测
3. **翻车展示**（1分钟）：Agent 是否混淆本月和本年收益率？是否只谈收益不谈风险？
4. **成果展示**（1分钟）：组合月报，重点看归因分析和调整建议是否有数据支撑
5. **一句话总结**（30秒）
