# S7: 政策影响评估

> **类型**：产业/通用
> **难度**：★★★
> **建议时长**：90分钟
> **适合人群**：对政策敏感的行业（如新能源、医药、科技、金融）

> **📌 教学材料声明**：本任务卡使用模拟数据，所有公司名称、财务数据、市场信息均为教学虚构，不构成任何投资建议。

---

## 5分钟速用框

| 项目 | 建议 |
|------|------|
| **适用场景** | 政策影响评估、战略风险分析、区域布局讨论 |
| **推荐 Context** | `data/S7-material-A.md` 至 `data/S7-material-D.md` 四份政策与公司背景材料 |
| **Harness 起点** | 只基于已发布政策文本；区分已确认政策和待定提案；禁止过度解读 |
| **最小成果** | 政策摘要 + 直接/间接影响 + 机会/风险 + 应对建议 + 不确定性说明 |
| **汇报重点** | 展示 Agent 是否混淆政策对象，是否把“鼓励”说成“强制” |

## 背景设定

你是某跨国企业的战略部高级经理。公司正在评估在东南亚（泰国/越南/印尼）投资建设新能源汽车电池工厂的可行性。

最近，**欧盟宣布对中国电动汽车加征临时反补贴税**，同时**美国《通胀削减法案》（IRA）对电池供应链的本土化要求**也在收紧。你需要快速评估这些政策变化对公司战略的影响。


---

## 任务目标

基于提供的政策材料，完成一份**政策影响评估报告**：

1. **政策要点摘要**（核心条款）
2. **直接影响**（对目标行业/企业的即时影响）
3. **间接影响**（对上下游、竞争格局的中长期影响）
4. **机会分析**（政策可能带来的新机会）
5. **风险分析**（政策可能带来的新风险）
6. **应对建议**

## 提供的材料

### 材料A：欧盟反补贴税政策

```
【欧盟对中国电动汽车反补贴税政策（2024年10月）】

政策要点：
- 欧盟委员会宣布对中国产电动汽车征收临时反补贴税
- 税率：
  - 比亚迪：17%
  - 吉利：18.8%
  - 上汽：35.3%
  - 配合调查的其他企业：20.7%
  - 未配合调查的企业：35.3%
- 特斯拉等在中国生产的车企，如申请单独审查，可获得单独税率
- 临时税自2024年10月30日起征收，为期4个月
- 最终税率将在2024年11月投票决定，若通过则征收5年

背景：
- 欧盟称中国电动汽车受益于"不公平补贴"（包括政府补贴、低息贷款、原材料优惠等）
- 2023年中国电动汽车占欧盟电动汽车进口市场份额约8.2%
- 欧盟本土车企（大众、Stellantis等）对此表示支持

中方回应：
- 中国商务部表示强烈反对，称将采取必要措施维护中国企业权益
- 中国机电商会代表12家车企向欧盟法院提起诉讼
```

### 材料B：美国IRA法案电池本土化要求

```
【美国《通胀削减法案》（IRA）电池供应链要求】

现有要求（2024年）：
- 电动汽车要获得$7,500税收抵免，需满足：
  - 电池关键矿物（锂、钴、镍等）40%来自美国或自贸协定国
  - 电池组件50%在北美制造
- 上述比例每年递增10%，2029年达到100%

政策影响：
- 中国企业若要在美国市场获得税收抵免，需在北美或自贸协定国建厂
- 目前中国电池企业在美直接设厂面临政治阻力
- 部分企业通过技术授权、合资等方式间接进入

最新动态：
- 2024年10月，美国国会某议员提出修正案，
  拟进一步收紧"外国敏感实体"（包括部分中国企业）的税收抵免资格
- 该修正案尚未进入投票程序
```

## 输出要求

### 格式要求

1. **政策要点摘要**（200字以内，核心条款）
2. **直接影响**（对公司的即时影响，2-3条，bullet points）
3. **间接影响**（对上下游、竞争格局的中长期影响，2-3条）
4. **机会分析**（2条，政策可能带来的新机会）
5. **风险分析**（2条，政策可能带来的新风险）
6. **应对建议**（3条，每条≤50字）
7. **不确定性说明**（政策执行细则尚未明确的部分）

### 约束要求

- 只基于已发布的政策文本，不猜测未发布的细则
- 对政策解读存在分歧的地方，列出不同观点
- 禁止过度解读（如把"鼓励"说成"强制"）
- 完成后列出2个政策执行中最大的不确定性

## 建议的 Prompt 框架

```
【Context】
我们是某中国电池企业的战略部，正在评估东南亚建厂的可行性。
已上传欧盟反补贴税政策、美国IRA法案要求、东南亚市场环境和公司背景。

【Objective】
基于提供的材料，生成一份政策影响评估报告。

【Style】
战略咨询风格，宏观视角与微观执行结合。

【Tone】
客观、冷静，承认政策不确定性。

【Audience】
公司战略委员会（CEO、CFO、战略VP），他们需要基于评估做出投资决策。

【Response】
1. 政策要点摘要（200字）
2. 直接影响（2-3条）
3. 间接影响（2-3条）
4. 机会分析（2条）
5. 风险分析（2条）
6. 应对建议（3条，≤50字/条）
7. 不确定性说明

【Constraint】
- 只基于已发布的政策文本
- 不猜测未发布的细则
- 政策解读分歧处列出不同观点
- 禁止过度解读
- 完成后列出2个最大不确定性
```

## 评价要点

| 维度 | 重点 |
|------|------|
| **Prompt设计** | 是否要求了机会+风险+建议？是否定义了Audience？ |
| **Context管理** | 是否综合使用了三个政策+市场环境+公司背景？ |
| **Harness约束** | 是否有"不猜测细则"的约束？是否有不确定性说明？ |
| **输出质量** | 评估是否全面？建议是否可行？ |
| **迭代过程** | 是否发现了政策之间的矛盾？如何处理？ |


---

## 提示与技巧

### 可能的"翻车"点

1. **Agent 可能把"临时"说成"永久"**：欧盟目前是"临时税"，Agent可能忽略"4个月"和"可能投票否决"
2. **Agent 可能混淆政策主体**：把欧盟政策和IRA混为一谈
3. **Agent 可能过度悲观**：看到关税就说"不能进入"，忽略东南亚的机会

4. **如果 Context 只给了政策文本没给公司背景**：Agent 无法分析"对公司的影响"，只能做通用政策解读
5. **如果只给了东南亚市场数据没给欧盟和IRA政策**：分析会变成"东南亚市场不错"的乐观结论，忽略最大外部风险
### 如何提升质量

- 明确要求"区分已确认的政策和待定的提案"
- 要求"对每个政策标注时间线（何时生效、何时到期、何时投票）"
- 增加约束："分析需平衡正面和负面影响"

### 可扩展的方向

- 增加"情景分析"（乐观/中性/悲观三种情景）
- 增加"竞争对手应对"（同行怎么做的？）
- 增加"时间轴"（未来12个月的关键政策节点）

## 汇报提示

汇报时建议结构：
1. **场景与目标**（30秒）：选了S7政策影响评估，目标是评估欧盟关税和美国IRA对东南亚建厂的影响
2. **Prompt 展示**（1分钟）：重点说明如何用 Constraint 区分已确认政策vs待定提案、禁止过度解读
3. **翻车展示**（1分钟）：Agent 是否把"临时税4个月"说成永久？是否只看风险忽略东南亚机会？
4. **成果展示**（1分钟）：政策影响评估报告，重点看不确定性和机会分析
5. **一句话总结**（30秒）
