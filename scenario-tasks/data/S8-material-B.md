# S8 材料B：本公司产品定位

【本公司产品定位（模拟）】

目标市场：20-30万元智能电动轿车
目标用户：一二线城市年轻家庭（30-40岁，首购或换购）

产品定义：
- 续航目标：CLTC 650-750km
- 智驾目标：高速NOA标配，城市NOA选装
- 座舱目标：高通8295芯片，支持多屏联动
- 空间目标：轴距≥2950mm，后排可放儿童座椅+行李
- 充电目标：兼容国标，自建超充网络（3年内500座）
- 品牌定位：科技感+家庭友好，"聪明且可靠"

核心差异化假设：
- 假设1：家庭场景智能化（儿童模式、宠物模式、露营模式）
- 假设2："透明定价"——无选装包，标配即满配
- 假设3：超长质保——三电终身质保
