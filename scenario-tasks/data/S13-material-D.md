# S13 材料D：客户背景

【客户背景：某跨国制造企业】

现有业务：汽车零部件（座椅、内饰、电子件）
规模：全球营收约50亿美元，员工3万人
现有市场：中国（40%）、欧洲（30%）、北美（20%）、其他（10%）

进入动机：
- 中国汽车市场增长放缓，需寻找新增长市场
- 现有客户（车企）正在东南亚建厂，有配套需求
- 公司有一定的制造能力和供应链管理经验

资源：
- 资金：可投入2-5亿美元
- 制造：有柔性制造能力，可快速调整产线
- 客户：与多家全球车企有长期合作关系

短板：
- 无电动车核心零部件（电池、电机、电控）技术
- 对东南亚市场了解有限
- 本地供应链尚未建立
