# S6 材料A：公司公开信息摘要

【宇树科技 公开信息摘要】

公司概况：
- 成立时间：2016年
- 总部：杭州
- 创始人：王兴兴（上海大学硕士，机械工程专业）
- 员工：约300人（其中研发占比60%）

产品：
1. Go系列（四足机器人/机器狗）：
   - Go1：消费级，售价1.6万元起，面向教育、娱乐
   - Go2：行业级，售价约5-10万元，面向巡检、安防
   - 负载能力：Go2可承载约10kg物品

2. H系列（人形机器人）：
   - H1：2023年发布，售价约65万元，面向科研、展览
   - G1：2024年发布，售价约9.9万元，面向教育、轻工业

3. 配件与软件：
   - 电池、充电器、摄像头模组
   - 软件开发包（SDK）、仿真平台

客户：
- 消费级：极客、科技爱好者、教育机构（占比约20%）
- 行业级：电力巡检（国家电网等）、安防（公安/园区）、物流（仓库巡检）
- 科研级：高校实验室、AI研究机构
- 海外客户：欧美、日韩、中东（占比约30%）

销售渠道：
- 官网直销（主要渠道）
- 京东/天猫旗舰店（消费级产品）
- 行业客户：直销团队+代理商
- 海外：通过当地代理商和展会

收入模式：
- 硬件销售（一次性）：约占80%
- 软件授权/订阅（年度）：约占10%
- 配件与服务：约占10%

合作伙伴：
- 芯片：英伟达（Jetson平台）、Intel
- 传感器：Intel（RealSense深度相机）
- 供应链：杭州本地精密制造供应商
- 渠道：京东、部分海外代理商

竞争：
- 国内：云深处科技、小米（CyberDog）、腾讯（Max）
- 国外：Boston Dynamics（Spot）、ANYbotics

融资：
- 2024年完成B2轮融资，估值约80亿元
- 投资方包括：红杉中国、顺为资本、深创投等
