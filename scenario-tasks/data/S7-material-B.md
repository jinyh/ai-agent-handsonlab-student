# S7 材料B：美国IRA法案电池本土化要求

【美国《通胀削减法案》（IRA）电池供应链要求】

现有要求（2024年）：
- 电动汽车要获得$7,500税收抵免，需满足：
  - 电池关键矿物（锂、钴、镍等）40%来自美国或自贸协定国
  - 电池组件50%在北美制造
- 上述比例每年递增10%，2029年达到100%

政策影响：
- 中国企业若要在美国市场获得税收抵免，需在北美或自贸协定国建厂
- 目前中国电池企业在美直接设厂面临政治阻力
- 部分企业通过技术授权、合资等方式间接进入

最新动态：
- 2024年10月，美国国会某议员提出修正案，
  拟进一步收紧"外国敏感实体"（包括部分中国企业）的税收抵免资格
- 该修正案尚未进入投票程序
