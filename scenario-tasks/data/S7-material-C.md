# S7 材料C：东南亚市场环境

【东南亚新能源汽车市场环境】

泰国：
- 政策：电动汽车进口关税减免（2024-2025年），
  本土化生产可获额外补贴（每辆约7万泰铢）
- 市场：2024年电动车销量约8万辆（同比+150%）
- 已有中国企业投资：比亚迪、长安、哪吒在泰建厂
- 电池：目前主要依赖进口，本土产能极少

越南：
- 政策：电动车进口关税较低（约10-15%），
  计划2030年禁售燃油车
- 市场：2024年电动车销量约3万辆
- 已有投资：VinFast（本土品牌）占主导，部分中国零部件企业入驻

印尼：
- 政策：镍矿资源丰富，政府要求电动车本地化率40%以上
- 市场：2024年电动车销量约2万辆
- 已有投资：宁德时代、LG新能源在印尼投资镍矿/电池项目
