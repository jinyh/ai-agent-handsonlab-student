# S3 场景C：贵州茅台（备选）

【模拟新闻/公告材料：贵州茅台（过去30天）】

2024-09-22：茅台宣布与瑞幸咖啡再次联名，推出"酱香巧克力"。
               首日销量突破100万杯，社交媒体话题热度高。

2024-09-29：媒体报道称，飞天茅台批发价从月初的2800元/瓶
               下跌至2650元/瓶，引发市场对需求的担忧。
               公司回应称"价格波动属正常现象，公司生产经营正常"。

2024-10-01：茅台公布前三季度业绩预告：营收同比增长16%，
               净利润同比增长18%，符合市场预期。

2024-10-08：某财经自媒体发文称，年轻消费者对白酒的兴趣下降，
               茅台的"年轻化"战略效果存疑。文章阅读量10万+。

2024-10-12：茅台宣布推出"茅台冰淇淋"新款，定价66元/盒，
               在部分城市试点销售。市场反应平淡。

2024-10-15：某券商研报将茅台目标价从2300元下调至2100元，
               理由是需求端复苏不及预期，批价持续承压。

2024-10-18：媒体报道称，茅台正在开发数字化营销平台，
               计划通过D2C模式直接触达消费者，减少渠道层级。

2024-10-22：社交媒体上出现关于"茅台库存高企"的讨论，
               称部分经销商库存超过3个月。公司未公开回应。
