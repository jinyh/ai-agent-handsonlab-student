# S13: 调研报告生成

> **类型**：通用/产业
> **难度**：★★★★
> **建议时长**：90分钟
> **适合人群**：咨询、研究、战略背景，喜欢挑战

> **📌 教学材料声明**：本任务卡使用模拟数据，所有公司名称、财务数据、市场信息均为教学虚构，不构成任何投资建议。

---

## 5分钟速用框

| 项目 | 建议 |
|------|------|
| **适用场景** | 市场进入调研、咨询报告初稿、战略委员会材料 |
| **推荐 Context** | `data/S13-material-A.md` 至 `data/S13-material-D.md` 四份市场、政策、竞争和客户背景材料 |
| **Harness 起点** | 只基于给定材料；每条数据标注来源；不输出具体投资金额或回报预测 |
| **最小成果** | 执行摘要 + 市场/竞争/政策/壁垒 + 机会风险 + 进入策略 + 信息缺口 |
| **汇报重点** | 展示 Agent 是否混淆国家政策，建议是否结合客户资源和短板 |


---

## 背景设定

你是某咨询公司产业研究部的项目经理。你的客户（一家跨国制造企业）正在评估是否进入**东南亚新能源汽车市场**。

你需要基于有限的公开信息，快速生成一份**市场进入调研报告**，为客户的战略决策提供依据。


---

## 任务目标

基于提供的模拟材料，生成一份**结构化调研报告**，包含：

1. **执行摘要**（300字，结论先行）
2. **市场概览**（规模、增长、关键数据）
3. **竞争格局**（主要玩家、市场份额、战略动向）
4. **政策环境**（各国政策、补贴、关税、本地化要求）
5. **进入壁垒**（技术、资金、渠道、政策）
6. **机会与风险**
7. **进入策略建议**
8. **信息缺口清单**


---

## 输出要求

### 格式要求

1. **执行摘要**（300字，结论先行）
2. **市场概览**（表格：六国销量、渗透率、预测）
3. **竞争格局**（主要玩家，bullet points，附战略动向）
4. **政策环境**（表格：各国政策对比）
5. **进入壁垒**（技术、资金、渠道、政策，每项1-5星评分）
6. **机会分析**（2-3条）
7. **风险分析**（2-3条）
8. **进入策略建议**（3条，每条≤50字）
9. **信息缺口清单**（决策需要但报告缺少的信息）

### 约束要求

- 只基于提供的材料
- 每条数据标注来源
- 数据无法确认时标注[待核实]
- 禁止给出具体投资金额或回报预测
- 建议需有明确的逻辑支撑
- 完成后列出3个最关键的信息缺口

## 建议的 Prompt 框架

```
【Context】
我们是某咨询公司产业研究部，客户是一家跨国制造企业（汽车零部件，
全球营收50亿美元），计划进入东南亚新能源汽车市场。
已上传市场数据、政策环境、竞争格局和客户背景。

【Objective】
基于提供的材料，生成一份市场进入调研报告。

【Style】
咨询公司研究报告风格，数据驱动、逻辑清晰、图表化呈现。

【Tone】
客观、务实，承认信息局限。

【Audience】
客户企业的战略委员会，他们需要基于报告做出投资决策。

【Response】
1. 执行摘要（300字）
2. 市场概览（表格）
3. 竞争格局（bullet points）
4. 政策环境（表格）
5. 进入壁垒（1-5星评分）
6. 机会分析（2-3条）
7. 风险分析（2-3条）
8. 进入策略建议（3条，≤50字/条）
9. 信息缺口清单

【Constraint】
- 只基于提供的材料
- 数据标注来源
- 无法确认的数据标注[待核实]
- 禁止给出具体投资金额或回报预测
- 建议需有逻辑支撑
- 完成后列出3个关键信息缺口
```

## 评价要点

| 维度 | 重点 |
|------|------|
| **Prompt设计** | 是否要求了完整的报告结构？是否有信息缺口清单？ |
| **Context管理** | 是否综合使用了市场、政策、竞争、客户四份材料？ |
| **Harness约束** | 是否有"不预测回报"的约束？建议是否有逻辑支撑？ |
| **输出质量** | 报告是否专业？建议是否可行？ |
| **迭代过程** | 是否发现了材料之间的矛盾？如何处理？ |


---

## 提示与技巧

### 可能的"翻车"点

1. **Agent 可能混淆国家**：把泰国政策和印尼政策混在一起
2. **Agent 可能过度乐观**：看到增长快就说"机会巨大"，忽略竞争和政策风险
3. **Agent 可能脱离客户背景**：给出通用的进入建议，不考虑客户的汽车零部件背景

4. **如果 Prompt 没要求"对每个国家单独分析"**：Agent 可能把6国政策混为一谈，无法看出各国差异
5. **如果只给市场数据不给政策环境**：Agent 无法分析"政策壁垒"，进入策略建议会缺少关键约束
### 如何提升质量

- 要求"对每个国家单独分析，不要混为一谈"
- 要求"机会和风险必须平衡呈现"
- 增加约束："所有建议必须基于客户的具体资源与短板"

### 可扩展的方向

- 增加"情景分析"（乐观/中性/悲观三种情景下的市场规模）
- 增加"时间轴"（未来3年的关键节点）
- 增加"竞争对手反应预测"（如果客户进入，竞品会怎么应对？）

汇报时建议结构：
1. **场景与目标**（30秒）：选了S13调研报告，目标是生成东南亚新能源汽车市场进入调研报告
2. **Prompt 展示**（1分钟）：重点说明如何用 Constraint 禁止投资金额预测、要求数据标注来源
3. **翻车展示**（1分钟）：Agent 是否混淆泰国和印尼政策？是否脱离客户背景给出通用建议？
4. **成果展示**（1分钟）：调研报告，重点看信息缺口清单是否诚实
5. **一句话总结**（30秒）
