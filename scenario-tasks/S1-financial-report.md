# S1: 财报解读与同业对比

> **类型**：金融
> **难度**：★★★
> **建议时长**：90分钟
> **适合人群**：有财务分析背景，熟悉财务指标
> **📌 教学材料声明**：本任务卡使用模拟数据，所有公司名称、财务数据、市场信息均为教学虚构，不构成任何投资建议。

---

| 项目 | 建议 |

## 背景设定

你是某私募基金投研团队的分析师。投资委员会要求你快速评估**宁德时代（300750.SZ）**2024年第三季度的投资价值，并与**比亚迪**、**LG新能源**进行横向对比。

你的时间很紧：投委会15分钟后开会，你需要在3分钟内让委员们抓住核心结论。

---

## 任务目标

基于提供的模拟财报数据，完成以下工作：

1. **提取核心财务指标**（宁德时代2024Q3）
2. **同业横向对比**（宁德时代 vs 比亚迪 vs LG新能源）
3. **识别投资亮点与风险**
4. **生成投资价值摘要**（150字以内，面向投委会）

---

## 输出要求

### 格式要求

1. **核心指标对比表**（Markdown表格）
   - 列：指标 | 宁德时代 | 比亚迪 | LG新能源 | 说明
   - 行：营业收入、归母净利润、毛利率、净利率、经营性现金流、ROE、资产负债率

2. **积极变化**（2条，bullet points，每条不超过40字）

3. **关注风险**（2条，bullet points，每条不超过40字）

4. **投资价值摘要**（150字以内，面向投委会）

### 约束要求

- 只基于提供的模拟数据，不补充外部信息
- 如果某项指标未提供，标注[数据未提供]
- 不要给出买入/卖出建议，只提供分析
- 对同比/环比变化异常（>±50%）的指标，标注[变化显著]

## 提示与技巧

### 可能的"翻车"点

1. **Agent 可能混淆单位**：宁德和比亚迪是人民币，LG是美元——需要在 Prompt 中明确说明
2. **Agent 可能过度解读**：看到LG亏损就说"行业整体下滑"，忽略宁德和比亚迪的增长
3. **Agent 可能遗漏关键维度**：如只对比了利润，没对比现金流质量

4. **如果不加 Audience**：Agent 会用 Wikipedia 式的学术语气写，而不是面向投委会的3分钟速读格式
5. **如果只给一份财报而没给对比公司的数据**：Agent 只能分析宁德时代，无法横向对比，产出价值大幅降低
### 如何提升质量

- 在 Prompt 中明确要求"逐项对比，不要跳过任何指标"
- 要求 Agent "对每家公司的优势和劣势分别说明"
- 增加校验机制："完成后请检查是否有指标被遗漏"

### 可扩展的方向

如果时间充裕，可以尝试：
- 增加更多对比维度（如客户集中度、产能利用率、研发投入强度）
- 生成可视化图表建议（如柱状图、雷达图）
- 做敏感性分析（"如果毛利率下降2pct，对净利润的影响是..."）

1. **场景与目标**（30秒）：选了S1财报解读，目标是横向对比三家公司Q3财务表现
2. **Prompt 展示**（1分钟）：展示最终版 Prompt，重点说明 Audience 和 Constraint
3. **翻车展示**（1分钟）：Agent 有没有混淆人民币和美元单位？有没有过度解读LG亏损？
4. **成果展示**（1分钟）：对比表 + 投资价值摘要
5. **一句话总结**（30秒）
